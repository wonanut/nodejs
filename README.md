# 基于WebSocket的`方格游戏(online)`

该项目主要基于Node.js以及Vue开发完成，当前版本能够支持多人同时在线游玩、游戏匹配、消息通知等功能。

* 客户端和服务器通信使用WebSocket实现
* 客户端使用Vue+Element UI+Canvas实现
